## 🔧 Mobile Editing Apps (Post-Processing)

After generating images, you may need to:
- Remove backgrounds (make transparent)
- Resize to exact dimensions
- Reduce colors to 4-color palette
- Fix artifacts

### Free Mobile Apps:

1. **Background Eraser** (Android/iOS)
   - Remove white backgrounds → make transparent PNG

2. **Pixel Studio** (Android/iOS) - FREE
   - Edit pixel art directly
   - Resize, recolor, fix pixels
   - Export as PNG with transparency

3. **PicsArt** (Android/iOS) - FREE with watermark
   - General image editing
   - Remove backgrounds, resize

4. **PhotoRoom** (Android/iOS) - FREE
   - Excellent automatic background removal
   - Batch processing available

5. **Canva** (Android/iOS) - FREE
   - Resize images to exact pixel dimensions
   - Simple editing tools

## 📋 Step-by-Step Mobile Workflow

### For Each Asset Type:

**Step 1: Generate**
1. Open your chosen AI generator app
2. Copy & paste the relevant prompt from above
3. Generate 4-8 variations
4. Select the best one
5. Download to your phone

**Step 2: Clean Up**
1. Open "PhotoRoom" or "Background Eraser"
2. Remove white background → save as transparent PNG
3. If needed, open in "Pixel Studio" to fix artifacts

**Step 3: Resize & Optimize**
1. Open "Pixel Studio" or "Canva"
2. Resize to exact dimensions:
   - Tiles: 32×32px
   - Player sprites: 32×32px
   - Numa sprites: 56×56px
   - Battle backgrounds: 160×144px
3. Reduce colors to 4 shades if needed
4. Export as PNG

**Step 4: Transfer to Project**
1. Upload to GitHub via mobile browser OR
2. Email to yourself and download on computer OR
3. Use GitHub mobile app to commit directly OR
4. Use cloud storage (Google Drive, Dropbox) as intermediary

## 💡 Pro Tips for Mobile Generation

1. **Batch Generate**: Create all tiles in one session for consistency
2. **Save Prompts**: Keep this guide bookmarked for quick copy-paste
3. **Use Same Seed**: If your AI tool supports seeds, use the same seed for consistent style
4. **Name Files Immediately**: Rename downloads to match code expectations:
   - `grass.png`, `water.png`, `player_down_1.png`, etc.
5. **Test Frequently**: Upload each asset and test in-game before generating more

## 🆘 Troubleshooting

**Problem**: AI generates non-pixel art
- **Solution**: Add "pixel art" at the start of every prompt, specify "8-bit" or "16-bit"

**Problem**: Colors don't match Game Boy palette
- **Solution**: Add exact hex codes to prompt: `#9bbc0f #8bac0f #306230 #0f380f`

**Problem**: Background isn't transparent
- **Solution**: Use PhotoRoom or Background Eraser app first

**Problem**: Wrong dimensions
- **Solution**: Use Pixel Studio or Canva to resize to exact px

**Problem**: Inconsistent art style between assets
- **Solution**: Use same AI tool + same prompt structure + similar seeds
